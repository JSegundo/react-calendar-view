Roadsurfer dashboard

## SETUP

### clone the repo

```
git clone https://github.com/JSegundo/roadsurfer-calendar.git
```
### install dependencies

```
npm install
```

### setup .env file

VITE_API_URL=https://605c94c36d85de00170da8b4.mockapi.io/

### run on dev mode

```
npm run dev
```

### run tests
``` npm run test```
